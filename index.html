Welcome

This website was created to provide a more comprehensive documentation of the dlls used by MB 2 Bannerlord. All entries contain a practical example to further show how certain features are used.

Getting started

To get started you can use this link if you work on Windows. Should you use Linux I've made this Tutorial on how you can setup your modding environment on there. Keep in mind though that you won't be able to perform any OS specific operations like reading/writing files or fetching all files in a directory as MB 2 Bannerlord was made using the .NET Framework 4.7.2 and you're going to compile with .NET Core 6.0+. This will cause a version mismatch on OS operations. Making logical changes like adding listeners, adding buttons/menus is all fine though.
